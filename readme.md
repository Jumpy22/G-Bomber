# G-Bomber
A Gmail-based email bomber

## Features
 - [x] Simple Installation
 - [x] Send 25+ MPM (Messages Per Minute)
 - [x] HTML Support
 - [ ] Multiple SendTo Support (Upcoming)
 - [ ] Multiple Sender Support (Upcoming)
 - [ ] Proxy Support (Upcoming)

## Installation

    pip install requests
    python main.py
